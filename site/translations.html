Although we don't develop a commercial or professional (whatever that means!) product, we put a lot of effort and time to make the menus and options self-explanatory. Nevertheless, all application messages and documentation texts are written in English. This might cause trouble to non-English speaking users. So, if you really want to help SocNetV evolve and/or contribute something back to you favorite free software, you can start translating this application!
Translating SocNetV means you have two options:
  • Translate the GUI
    At the moment, there are some 600 GUI messages. All these are saved for translation in a text file. Download this file (~200K), open it with a text editor and start translating: you read the message inside the source tags and write its translation below inside the translation tag:

    <source>Creates a new network</source>
    <translation type="unfinished"> TRANSLATION HERE</translation>

    When you finish some messages (you don't have to translate all at once you know!), save the file and send it back to us.
  • Translate the Manual
    The most recent version of the SocNetV manual is here. You can just download the .html files, translate them and send the translated ones back to us.
You can opt to translate the GUI, the manual or both.
In any case, you will win your place in the SocNetV "hall of fame"... :)
