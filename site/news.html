Feb 27, 2014
SocNetV version 1.0 has just been released! It is our first version based on the new Qt5 toolkit and features PageRank calculation and layout as well as various bug fixes. Source code available in the Downloads menu. Have fun!

Oct 13, 2013
There is a new SocNetV version coming up. It is based on Qt5 toolkit and it will be the first ever 1.x version of the project! SocNetV version 0.90 was the final Qt4 version. From now on, development will continue only in 1.x series and Qt5. So, if you want to follow SocNetV in the future, it's time to install Qt5 on your computer! Also, source code and version control management have been moved to Git. You can get a development shapshot from here

Dec 22, 2012
After a long inactivity we are happy to announce that SocNetV now includes technical documentation for both users and developers. Vaggelis Motesnitsalis, undergraduate in Computer Science, was kind enough to prepare and contribute a complete Software Requirements Specification (SRS, see Wikipedia) document in PDF format for Social Network Visualizer. The PDF is already in trunk and will be included in the next version 0.91 of SocNetV, due to be released at some point in 2013.

Oct 13, 2010
SocNetV version 0.90 has been released. This is a major release with new feautures (such as Information and Power Centrality) as well as many bug fixes. See full changeLog. Source code and binary packages available in the Downloads menu. Have fun!
