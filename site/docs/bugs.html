Known Bugs

The network file parser is not 100% trustworthy. For instance, complicated Pajek or GraphML files can crash the application.

Antialiasing is off by default to boost the speed of node movement. You can enable antialising by pressing F8.

Report a bug

Report bugs to dimitris.kalamaras@gmail.com.
Please, to send a meaningful bug report, run socnetv from a terminal, press F9 and repeat the steps which lead to bug manifestation. SocNetV will be printing debug messages on the standard output -- the terminal in Linux.
