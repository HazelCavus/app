To compile or install SocNetV, you need Qt 4.4 and QtWebKit development files. For instance, in openSUSE 11+ and Debian-based distros, install libQtWebKit-devel and libqt4-webkit, respectively.

There are four ways to install SocNetV:
A. Compile from Source code
  • Latest version: SocNetV-0.50.tar.gz

    Click on the link above and download the archive with the source code. Then decompress it, and compile it with these commands:
    tar zxfv SocNetV-0.XX.tar.gz
    cd socnetv
    ./configure
    make
    sudo make install
    In the end, run SocNetV with:
    socnetv
B. Install from Binary packages
  • For Ubuntu/Kubuntu

    All you have to do is add the following lines in your /etc/apt/sources.list file:

    deb http://ppa.launchpad.net/dimitris-kalamaras/ppa/ubuntu intrepid main
    deb-src http://ppa.launchpad.net/dimitris-kalamaras/ppa/ubuntu intrepid main

    Then save it, exit the text editor, and type in:

    sudo apt-get update
    sudo apt-get install socnetv

    This repository is signed with 61AE869C37A4FCC5A73FD02EE088941209CFE071 OpenPGP key. See instructions to add the key to your system, if you like.


  • For Debian & Debian-derived distros

    An older version of SocNetV is in the 'experimental' repository. Add the line

    deb http://ftp.debian.org/debian/ experimental main

    to your sources.list; save it, then type in:

    sudo apt-get update
    sudo apt-get install -t experimental socnetv


  • For Gentoo builders

    Markos Chandras (hwoarang) added SocNetV into the 'qting-edge' overlay, which also houses new Qt4 and KDE4 software. To install the qting-edge overlay type in this command:

    layman -a qting-edge
C. Development version
  • If you want to test the latest/current development version of SocNetV, check it out using this command (you need to have installed the Subversion package):

    svn co http://socnetv.svn.sourceforge.net/svnroot/socnetv/trunk

    Then, type the commands:

    cd trunk
    ./configure
    make
    socnetv

    Please note that this version is not always stable.
D. Klik-able setup
  • Finally, for Linux users who don't mind running an older version, you can try SocNetV without actually installing it. Just go to http://socnetv.klik.atekon.de/ and then click on the large blue "download" button (to the bottom of the page).
    Warning: klik service is not always updated with the latest version of SocNetV.
